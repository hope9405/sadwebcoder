# 2. rest API
## 2-1. rest API이란?
 rest란 "Representational State Transfer"의 약자로 표현에 의한 상태 전달을 의미한다. 여기서 표현은 이름을 상태는 정보를 뜻한다.   
 * 표현 -> 이름(ex: 학생의 학번을 'number'라는 이름으로 표현한다)
 * 상태 -> 정보(ex: 'number'라는 이름으로 표현된 정보의 값)
